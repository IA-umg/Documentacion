## ¿Qué es Koa?

Koa es una inteligencia artificial que permite a los usuarios realizar consultas relacionadas con SQL y obtener respuestas fundamentadas en una base de conocimientos.

El sistema ofrece una interfaz de tipo chat, donde el usuario puede escribir preguntas y recibir respuestas.

---

## Funcionalidades principales

- Realizar consultas sobre SQL
- Obtener respuestas generadas por Koa
- Visualizar la fuente donde se obtuvo la información
- Cargar archivos para incrementar la base de conocimientos

---

## Requerimientos

Para acceder a Koa, se necesita lo siguiente:

- Navegador moderno (Chrome, Firefox, Microsoft Edge)
- Conexión a internet
- Dispositivos compatibles (PC, Tablet o Smartphone)

---

## Inicio de Sesión

Para ingresar a la aplicación, el usuario debe autenticarse utilizando un correo y una contraseña registrada en el sistema.

- Campo de ingreso del correo electrónico del usuario
- Campo de ingreso de la contraseña del usuario
- Botón de inicio de sesión
- Botón de registro de nuevo usuario

---

## Registro de Usuario

Para registrarse como un nuevo usuario, se debe presionar el botón **Regístrate**, el cual redireccionará al formulario de registro. El usuario debe llenar todos los campos antes de poder registrarse dentro de la aplicación.

- Campo de ingreso del nombre del usuario
- Campo de ingreso del correo electrónico del usuario
- Campo de ingreso de contraseña
- Campo de ingreso de confirmación de contraseña
- Botón de creación de cuenta

---

## Conversaciones con Koa

Luego de iniciar sesión, se ingresará a la pantalla de chat, en la cual se puede comenzar a tener conversaciones con Koa.

- **Área de chat:** En esta área se podrán visualizar todas las preguntas y respuestas realizadas durante la sesión.
- **Campo de ingreso de mensaje**
- **Botón de selección de modelo:** Actualmente Koa ofrece la funcionalidad de seleccionar entre distintos modelos, ya sea Gemini o Grok.
- **Botón de carga de archivo:** Koa permite realizar la carga de archivos para aumentar la base de conocimientos.

---

## Carga de Archivos

Para realizar una carga de archivos, se debe presionar el botón **Alimenta a Koa**, esto habilitará un área en la cual el usuario puede cargar los archivos que desee para alimentar la base de conocimientos.

- **Botón de carga de archivo**
- **Área de carga de archivos:** En esta área el usuario puede arrastrar o seleccionar los archivos para alimentar la base de conocimientos.
- **Ventana de selección de archivos:** Ventana emergente en la cual se puede buscar y seleccionar los archivos.
- **Archivos cargados:** En esta sección se puede observar los archivos que se cargaron durante la sesión.

Luego de la carga exitosa, se pueden realizar preguntas sobre esta información; Koa los tomará en cuenta para responder.
